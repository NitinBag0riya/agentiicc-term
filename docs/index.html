User Management

POST {{baseUrl}}/user

Create User

Creates a new user and auto-sets the 'userId' collection variable.

Request Body (JSON)
{
  "telegramId": 999111,
  "username": "postman_app_user"
}
POST {{baseUrl}}/user/credentials

Link Credentials (Aster)

Links Aster credentials using apiKey and apiSecret.

Request Body (JSON)
{
  "userId": "{{userId}}",
  "exchange": "aster",
  "apiKey": "test-key-123",
  "apiSecret": "test-secret-456"
}
POST {{baseUrl}}/user/credentials

Link Credentials (Hyperliquid)

Links Hyperliquid credentials using address and privateKey.

Request Body (JSON)
{
  "userId": "{{userId}}",
  "exchange": "hyperliquid",
  "address": "0xUserAddress",
  "privateKey": "0xUserPrivateKey"
}
GET {{baseUrl}}/user/exchanges?userId={{userId}}

List Exchanges

Order Placement

POST {{baseUrl}}/order

1. LIMIT Order

Standard GTC Limit Order. Price set to 3000 (below market 3200).

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "BUY",
  "type": "LIMIT",
  "quantity": "0.01",
  "price": "3000",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

2. MARKET Order

Executes immediately at best available price

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "BUY",
  "type": "MARKET",
  "quantity": "0.005",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

3. IOC (Immediate Or-Cancel)

Limit order that must fill immediately or be cancelled.

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "BUY",
  "type": "LIMIT",
  "quantity": "0.01",
  "price": "3200",
  "timeInForce": "IOC",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

4. POST_ONLY

Limit order that acts as Maker only. Rejected if it crosses spread.

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "BUY",
  "type": "LIMIT",
  "quantity": "0.01",
  "price": "3000",
  "postOnly": true,
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

5. STOP_MARKET

Triggers a Market sell when price falls to triggerPrice. Should be below market (3200).

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "SELL",
  "type": "STOP_MARKET",
  "quantity": "0.01",
  "triggerPrice": "2900",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

6. STOP_LIMIT

Triggers a Limit sell at 'price' when price falls to 'triggerPrice'.

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "SELL",
  "type": "STOP_LIMIT",
  "quantity": "0.01",
  "triggerPrice": "2900",
  "price": "2850",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

7. TAKE_PROFIT_MARKET

Triggers a Market sell when price rises to triggerPrice.

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "SELL",
  "type": "TAKE_PROFIT_MARKET",
  "quantity": "0.01",
  "triggerPrice": "3500",
  "exchange": "{{exchangeId}}"
}
POST {{baseUrl}}/order

8. TRAILING_STOP (Aster Only)

Trailing stop with callback rate %

Request Body (JSON)
{
  "symbol": "ETHUSDT",
  "side": "SELL",
  "type": "TRAILING_STOP_MARKET",
  "quantity": "0.01",
  "callbackRate": "2",
  "exchange": "aster"
}
